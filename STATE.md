### Как проверить
- Сервер локально: `PORT=39xx node server.js`; `curl POST /register` → есть `sessionId`; `curl POST /screenshot` → файл в `screenshots/<id>/`.
- Прод: `GET https://remotedeskpbx-server.onrender.com/` → `"server":"express-v2"` (после деплоя проверить `/register` на `sessionId`).
- E2E (нужны 2 ПК): клиент шарит экран → в админке у сессии снимки с шагом ~30с; сотрудник пишет → у клиента всплывает окно чата; снимок сотрудника → файл в Документах; повторный запуск employee.exe → сразу рабочий экран.
